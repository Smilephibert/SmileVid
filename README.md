# SmileVid

SmileVid helps creators upload, edit, and share videos easily, built with Next.js and Tailwind CSS.

## Features

✅ Upload videos  
✅ Edit and manage content  
✅ Share with others

## Getting Started

1. Clone the repo:
   git clone https://github.com/Smilephibert/SmileVid.git

2. Install dependencies:
   npm install

3. Run locally:
   npm run dev
